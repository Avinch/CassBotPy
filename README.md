# Cassandra
[![license](https://img.shields.io/github/license/mashape/apistatus.svg?style=flat-square)]()

Named after the [Digital Haunt character](https://digitalhaunt.fandom.com/wiki/Cassandra), Cassandra is a Discord bot built to moderate the unofficial Area 11 Discord server. This bot has many different features and functions, including:

- Message archiving to backup channels and servers
- User self role assignment (`-role [add/remove] [rolename]`)
- Role mentionability management (`-role [enable/disable] [rolename]`) and role ping abuse prevention
- Suggestion intregration with GitHub (`-suggest [suggestion]` sends the suggestion to this repo's issues)
- General moderation/management tools (`-kick`, `-ban`, `-purge`, `-eval` etc.) 
- General information commands (`-help`, `-rules`, `-serverinfo` etc.)
- Fun commands (`-ask`, `-roll`, `-clap`, `-dog`, `-cat` + more)
